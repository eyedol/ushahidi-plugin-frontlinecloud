== Description ==
Receive Messages from a FrontlineSMS Installations.

== Installation ==
1. Copy the entire /frontlinecloud/ directory into your /plugins/ directory.
2. Activate the plugin.
3. Click on the [settings] link to get your Frontlinecloud URL
